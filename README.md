# Waterfall Task Planner (MVP)

https://chat.qwen.ai/c/0b2254c7-218e-4ff1-a2d1-c8a3d74bf0f1

Простой инструмент планирования задач по Waterfall-методологии, реализованный на HTML + CSS + JavaScript.

## Функциональность

- Управление исполнителями (имя, эффективность)
- Добавление задач с основными атрибутами
- Учет зависимостей между задачами
- Расчет дат с учетом зависимостей и блокеров
- Приоритезация задач
- Сохранение/загрузка в XML
- Современный адаптивный интерфейс

## Использование

1. Откройте `index.html` в браузере
2. Добавьте исполнителей
3. Создайте задачи с зависимостями
4. Нажмите "Рассчитать график"
5. Сохраните результат в XML

## Особенности

- Учет **рабочих дней** (без выходных)
- Визуальное выделение задач по приоритету
- Защита от циклических зависимостей
- Поддержка множественных исполнителей
- Автоматический расчет дат на основе зависимостей
- Современный UI с адаптивной версткой

## Возможности

### Исполнители
- Добавление/удаление исполнителей
- Настройка производительности (множитель)
- Статус доступности

### Задачи
- Название и описание
- Длительность в рабочих днях
- Даты начала/окончания
- Блокеры ("начать не раньше", "закончить не позже")
- Назначение исполнителей
- Связь с эпиками
- Бизнес-ценность (приоритет)
- Зависимости от других задач
- Комментарии

### Планирование
- Автоматический расчет графика
- Учет зависимостей между задачами
- Проверка на циклические зависимости
- Визуализация в таблице с цветовой индикацией приоритетов

### Экспорт/Импорт
- Сохранение в XML формат
- Загрузка из XML файла
- Перенос проектов между сессиями

## Технические особенности

- Использует только нативный JavaScript (без библиотек)
- Работает полностью в браузере (без сервера)
- Адаптивный дизайн для мобильных устройств
- Учет рабочих дней (понедельник-пятница)
- Защита от ошибок ввода данных

## Ограничения MVP

- Не поддерживает вложенные задачи (эпик/стори/таск)
- Нет диаграммы Ганта
- Нет мультипроектного режима
- Нет совместной работы нескольких пользователей
