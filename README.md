# Waterfall Task Planner (MVP with Enhanced Dependencies and Executors - Optimized)

Обновленный инструмент планирования задач по Waterfall-методологии с JSON-хранилищем, модальными окнами, компактной таблицей задач, сортировкой и фильтрацией, с отдельными окнами для управления зависимостями и исполнителями, с оптимизацией по доступности исполнителей.

## Функциональность

- Управление исполнителями через модальное окно
- Управление эпиками (название, описание, приоритет)
- Управление фичами (связь с эпиками, бизнес-ценность)
- Добавление/редактирование задач через модальное окно
- Учет зависимостей между задачами
- Расчет дат с учетом зависимостей и блокеров
- Приоритезация задач
- Сортировка таблицы задач по любому столбцу
- Фильтрация задач по исполнителям, фичам, эпикам, приоритетам
- Отдельные модальные окна для управления исполнителями и зависимостями задачи
- Наглядное отображение выбранных исполнителей и зависимостей
- Оптимизация расписания по доступности исполнителей (один исполнитель не может выполнять 2 задачи в один день)
- Сохранение/загрузка в JSON
- Современный адаптивный интерфейс с модальными окнами
- Компактная таблица задач

## Использование

1. Откройте `index.html` в браузере
2. Нажмите "Управление исполнителями" для добавления команды
3. Нажмите "Эпики" для создания эпиков
4. Нажмите "Фичи" для создания фич и связи с эпиками
5. Нажмите "Добавить задачу" для создания задач
6. Нажмите "Рассчитать" для автоматического расчета дат с оптимизацией по доступности исполнителей
7. Используйте панель фильтрации для поиска задач
8. Кликните по заголовку столбца для сортировки
9. Сохраните результат в JSON

- Оптимизация расписания по доступности исполнителей

- Возможность редактирования задач, эпиков и фич
- Компактная таблица задач с подсказками
- Сортировка по любому столбцу
- Фильтрация по нескольким критериям
- Хранение данных в JSON формате
- Отдельные окна для управления исполнителями и зависимостями

## Возможности

### Исполнители
- Добавление/удаление исполнителей
- Настройка производительности (множитель)
- Статус доступности
- Переключение доступности
- Фильтрация задач по исполнителям
- Управление исполнителями задачи через отдельное окно

### Эпики
- Добавление/удаление эпиков
- Название и описание
- Приоритет (1-9)
- Редактирование эпиков
- Фильтрация задач по эпикам

### Фичи
- Добавление/удаление фич
- Связь с эпиками (многие к одному)
- Бизнес-ценность (1-9)
- Описание фич
- Редактирование фич
- Фильтрация задач по фичам

### Задачи
- Название и описание
- Длительность в рабочих днях
- Даты начала/окончания
- Блокеры ("начать не раньше", "закончить не позже")
- Назначение исполнителей через отдельное окно
- Связь с фичами
- Бизнес-ценность (приоритет)
- Зависимости от других задач через отдельное окно
- Комментарии
- Редактирование задач

### Планирование
- Автоматический расчет графика
- Учет зависимостей между задачами
- Проверка на циклические зависимости
- Оптимизация по доступности исполнителей (один исполнитель не может выполнять 2 задачи в один день)
- Визуализация в таблице с цветовой индикацией приоритетов

### Сортировка и фильтрация
- Сортировка по любому столбцу (название, длительность, даты, исполнители, фичи, приоритет)
- Фильтрация по исполнителям
- Фильтрация по фичам
- Фильтрация по эпикам
- Фильтрация по приоритетам (высокий, средний, низкий)
- Сброс фильтров

### Хранение данных
- Хранение в формате JSON
- Сохранение всех данных: исполнители, эпики, фичи, задачи
- Загрузка проекта из JSON файла
- Перенос проектов между сессиями
